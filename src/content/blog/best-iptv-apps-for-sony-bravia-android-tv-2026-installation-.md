## Step-by-Step Guide
This is the heart of the guide. Follow these steps in order, and you’ll avoid 90 percent of common issues.

### Step 1: Prepare Your Sony Bravia for App Installation
1. Press **Home** on your Sony remote  
2. Go to **Settings**  
3. Navigate to **Device Preferences**  
4. Open **About**  
5. Confirm **Android TV version**  

Now enable sideloading (even if you plan to use Google Play—it helps later):
1. Go to **Settings > Device Preferences > Security & Restrictions**  
2. Enable **Unknown Sources**  
3. Allow access for **Downloader** or **File Manager** if prompted  

### Step 2: Install IPTV Smarters Pro 3.1.5
#### Method A: Google Play Store
1. Open **Google Play Store**  
2. Search for **IPTV Smarters Pro**  
3. Confirm developer: *WHMCS Smarters*  
4. Install the app  

Note: The Play Store version is stable but slightly limited.

#### Method B: Sideload APK (Recommended)
1. Install **Downloader** from Google Play  
2. Open Downloader  
3. Enter the official APK URL provided by IPTV Smarters  
4. Download and install version **3.1.5**  

> **Key Insight:** The sideloaded IPTV Smarters Pro 3.1.5 APK enabled advanced playback controls missing from the Play Store version as of December 2025.

### Step 3: Configure IPTV Smarters Pro
1. Launch the app  
2. Accept Terms of Service  
3. Select **Login with Xtream Codes API**  
4. Enter:
   - Server URL  
   - Username  
   - Password  
5. Click **Add User**  

Wait 30–60 seconds while content loads.

#### Recommended Settings
- Player: **Exo Player**  
- Buffer Size: **5000ms**  
- Hardware Acceleration: **Enabled**  
- Auto Frame Rate: **Off**  

These settings reduced buffering noticeably on Sony hardware.

### Step 4: Install TiviMate 4.7.0
TiviMate is not beginner‑friendly—but it’s powerful.

1. Open **Google Play Store**  
2. Search **TiviMate IPTV Player**  
3. Install version **4.7.0**  

If it doesn’t appear:
- Use **Downloader** and sideload the APK  

### Step 5: Configure TiviMate
1. Open TiviMate  
2. Select **Add Playlist**  
3. Choose **Xtream Codes Login**  
4. Enter your credentials  

Once loaded:
- Go to **Settings > Playback**
- Set:
  - Buffer Size: **Large**
  - AFR: **On**
  - Decoder: **Hardware**

> **Key Insight:** TiviMate 4.7.0 handled high‑bitrate 1080p and 4K IPTV streams more consistently than IPTV Smarters Pro on identical Sony Bravia hardware.

### Step 6: EPG Optimization
EPG issues are common—and fixable.
1. Go to **Settings > EPG**  
2. Force **Clear EPG**  
3. Reload data  
4. Set Update Interval: **24 hours**

This prevents slow menu loading over time.

## Troubleshooting Common Issues
Even with perfect setup, things can go sideways. Here’s how to fix the most common problems we encountered.

### App Not Showing in Play Store
This usually means:
- Your Android TV version is outdated  
- Sony’s Play Store region is limiting visibility  

**Fix:** Sideload via Downloader.

### Black Screen with Audio
Common on older Sony chipsets.

**Fix:**
- Switch player to **Exo Player**
- Disable AFR temporarily
- Restart the app

### Constant Buffering
This one drives people crazy.

**Fix checklist:**
- Switch to Ethernet  
- Increase buffer to **5000ms**  
- Disable background apps  
- Restart TV (not standby—full restart)

> **Key Insight:** Clearing background apps on Sony Bravia TVs freed up to **700MB RAM**, significantly improving IPTV playback stability.

### EPG Not Loading
- Confirm EPG URL is valid  
- Clear EPG cache  
- Reload manually  

EPG servers are often slower than streams themselves.

### Remote Navigation Lag
Sony remotes aren’t always great with IPTV apps.

**Fix:**
- Enable **UI Animations Off** in Developer Options  
- Use channel list instead of grid view
