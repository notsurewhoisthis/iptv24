Updates can change settings or break playback. Use this quick recovery list.

## Recovery checklist
1. Force quit and relaunch the app.
2. Clear cache and reload the playlist.
3. Re-enter credentials and EPG URL.
4. Reinstall the app if the issue persists.

## Version notes
If a new version breaks playback, note the version number and check our troubleshooting guide for updates or workarounds.
